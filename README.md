## :musical_note: A Simple Music Player

Create a virtual env

```
$ python3 -m venv myvenv
```

Initialize venv

```
$ source myvenv/bin/activate
```

Install requirements

```
$ pip install -r requirements.txt
```

Execute the script

```
$ python3 main.py
```
